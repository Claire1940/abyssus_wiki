<Section icon={Activity} title="How to Install abyssus mods Safely">

Installing mods for Abyssus is generally straightforward, but following a standardized process prevents file corruption and ensures compatibility between different tweaks. Most utility mods require you to place files within the game's installation directory, while trainers often run as separate executables.

<Steps>
  <Step num="1" title="Backup Your Save Files">
    Before installing any **abyssus mods**, locate your save folder (usually in AppData) and create a copy. This ensures you won't lose progress if a mod causes a crash.
  </Step>
  <Step num="2" title="Install ShaderToggler (If Required)">
    For visual mods like No-HUD, you must first install the ShaderToggler framework. Download the latest version and place it in the game's 'Binaries' folder.
  </Step>
  <Step num="3" title="Download and Extract Mod Files">
    Download your chosen mod from a reputable source like Nexus Mods. Extract the .zip or .rar contents directly into the specified game folder.
  </Step>
  <Step num="4" title="Verify Installation">
    Launch the game. For utility mods, check if the intro is skipped or if the HUD can be toggled via the assigned hotkey (usually 'Caps Lock' or a Function key).
  </Step>
</Steps>

<Callout type="info">
If the game fails to launch after installing a mod, use the Steam "Verify Integrity of Game Files" feature to reset your installation to the vanilla state.
</Callout>
